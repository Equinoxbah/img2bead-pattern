# Img2BeadPattern 拼豆图纸生成工具
## 项目简介
Python自动化工具，任意图片一键转换成**可打印拼豆网格图纸**，内置完整拼豆色卡，自动匹配色号、统计每种豆子用量，手工拼豆爱好者专用工具。

## 核心功能
1. 内置全套A/B/C/D/E/F/G/H/M/P/R/T/Q/Y/ZG拼豆色卡，精准RGB颜色匹配
2. 智能图片降噪分块采样，消除原图噪点，图纸颜色干净不杂乱
3. 自定义画布尺寸（默认48×48格子，可自行修改代码参数）
4. 生成带网格、格子内色号、右侧图例+用豆数量的高清打印图纸
